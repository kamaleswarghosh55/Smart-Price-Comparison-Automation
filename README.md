# Smart Price Comparison Automation

## Overview
A Selenium-based automation framework that compares product prices across live e-commerce platforms such as Flipkart, Amazon, and Croma.  
Designed with a clean structure, config-driven execution, and clear reporting, suitable for product-based QA automation roles.

---

## Highlights
- Automated price comparison on live websites
- Config-driven strategy to fetch:
  - Cheapest available price
  - First matching product price
- Excel execution report with status (PASS / PARTIAL / FAIL) and timestamps
- Screenshots captured for failed and partial scenarios
- Page Object Model (POM) based design
- Cross-browser execution support
- GitHub version-controlled project

---

## Tech Stack
Java | Selenium WebDriver | TestNG | Maven | Apache POI | Git | GitHub

---

## Execution
- Configure products and strategy in `config.properties`
- Run the `PriceComparisonTest` class using TestNG
- Reports and screenshots are generated automatically
